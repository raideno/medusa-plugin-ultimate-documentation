## Endpoint
**GET**: `<BACKEND_URL>/store/ultimate-entities/<ULTIMATE_ENTITY_ID>/documents/<DOCUMENT_ID>`


## Path Variables

- `<BACKEND_URL>`: your medusa backend URL.
- `<ULTIMATE_ENTITY_ID>`: the entity ID.
- `<DOCUMENT_ID>`: the entity document ID.

## Parameters

Nothing available.

## Return Value

```ts
{
  documents: {
    id: string;

    // other fields of your entity

    created_at: Date;
    updated_at: Date;
    deleted_at: Date;
  }
}
```
