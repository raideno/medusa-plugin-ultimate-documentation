# Instalation

Use one of the below commands to install the plugin into your medusa backend.

import { Tabs } from "nextra/components";

<Tabs items={["npm", "yarn", "pnpm"]} defaultIndex="1">
  <Tabs.Tab>
  ```bash copy

  ```
  </Tabs.Tab>
  <Tabs.Tab>
  ```bash copy
  yarn add medusa-plugin-ultimate
  ```
  </Tabs.Tab>
  <Tabs.Tab>
  ```bash copy
  pnpm i medusa-plugin-ultimate
  ```
  </Tabs.Tab>
</Tabs>
