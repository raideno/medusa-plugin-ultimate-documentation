# Configuration

In addition to this you'll need one of the env variables available:

- `MEDUSA_ADMIN_BACKEND_URL`: contain your backend URL.
- `BACKEND_URL`: contain your backend URL.

```ts filename="medusa-config.js" copy showLineNumbers
const plugins = [
  // other plugins
  /** @type {import('medusa-plugin-ultimate').Options} */
  {
    resolve: "medusa-plugin-ultimate",
    options: {
      enableUI: true,
      // IMPORTANT
      backendUrl: process.env.BACKEND_URL || "http://localhost:9000",
    },
  },
  // ...
];
```
