# String Array Type

Ideal for keywords and lists of strings, will be translated into a multi input keywords.

```ts copy showLineNumbers
import {
  UltimateEntityFieldTypes,
  UltimateEntityField,
} from "medusa-plugin-ultimate/dist/index";
```

```ts copy showLineNumbers
@UltimateEntityField({
    // required
    type: UltimateEntityFieldTypes.STRING_ARRAY,
    // options
    name?: string;
    description?: string;
    note?: string;
    group?: string;
})
@Column({ type: "string", array: true, })
keywords: string[];
```

## Parameters

- `type`: `UltimateEntityFieldTypes.STRING_ARRAY`,
- `name`: (optional) override field name, will be displayed on the UI as a label.
- `description`: (optional) to give more details about the field, will be displayed on the UI below the input.
- `note`: (optional) displayed as a tooltip when the infos icon is hovered.
- `group`: (optional) fields of the same group are displayed in the same block [(details)](/advanced/grouping).
