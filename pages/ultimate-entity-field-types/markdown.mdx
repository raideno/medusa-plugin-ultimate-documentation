# Markdown Array Type

Ideal for long formatted texts, will be translated into a rich / markdown text editor.

```ts copy showLineNumbers
import {
  UltimateEntityFieldTypes,
  UltimateEntityField,
} from "medusa-plugin-ultimate/dist/index";
```

```ts copy showLineNumbers
@UltimateEntityField({
    // required
    type: UltimateEntityFieldTypes.MARKDOWN,
    // options
    defaultValue: string;
    name?: string;
    description?: string;
    note?: string;
    group?: string;
})
@Column({ type: "varchar" })
content: string;
```

## Parameters

- `type`: `UltimateEntityFieldTypes.MARKDOWN`,
- `defaultValue`: (optional)
- `name`: (optional) override field name, will be displayed on the UI as a label.
- `description`: (optional) to give more details about the field, will be displayed on the UI below the input.
- `note`: (optional) displayed as a tooltip when the infos icon is hovered.
- `group`: (optional) fields of the same group are displayed in the same block [(details)](/advanced/grouping).

## Example Use Cases

This field can be used for multiple things

- Blog Post content
- Product Rich Description
